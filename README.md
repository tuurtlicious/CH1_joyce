# xt2_challenge1_ccj

Voor challenge 1 is er een Dashboard gemaakt voor SpaceX. Het dashboard laat aan de bemanning zien hoeveel voorraad er is van voedsel, hoeveel brandstof er nog is en hoe ver zij van de aarde vandaan zijn sinds de lancering.
Het dashboard toont in dit geval drie grafieken. Elk grafiek heeft zijn eigen vorm, een donut, een staafdiagram en een lijngrafiek. 

De donut grafiek is gekozen zodat zij in een oogopslag inzien hoeveel voorraad er nog over is. 
Dit is belangrijk want dan kunnen ze inschatten hoeveel er over is per persoon.

De staafdiagram toont aan hoeveel brandstof er over is tijdens de reis, zo kan de bemanning een schatting maken wanneer zij terug moeten naar de aarde omdat er dan niet meer genoeg brandstof over is.

De lijngrafiek laat zien hoever zij zich bevinden van de aarde, in combinatie met de brandstof en de afstand, kan de bemanning een schatting maken om de terugreis naar de aarde in te plannen.
